## 组件详细说明

### 1. 顶部工具栏
- **标题**: 显示"知识库健康仪表盘"图标和文字
- **刷新按钮**: 手动刷新数据（快捷键 F5）
- **导出按钮**: 导出当前状态为报告（开发中）

### 2. 关键指标卡片区（3 列）

#### 总知识量卡片
- **数值**: 125,432（大号粗体）
- **标签**: "总知识量"
- **趋势**: "↑ 持续增长"（绿色向上箭头）
- **边框色**: 绿色（#4CAF50）

#### 今日新增卡片
- **数值**: +342
- **标签**: "今日新增"
- **趋势**: "↑ 活跃更新"
- **边框色**: 蓝色（#2196F3）

#### 平均新鲜度卡片
- **数值**: 45 天
- **标签**: "平均新鲜度"
- **趋势**: "↓ 需要更新"（红色向下箭头）
- **边框色**: 橙色（#FF9800）

### 3. 健康分数仪表盘（左侧 1/3 宽度）

#### 视觉元素
- **半圆仪表盘**: 
  - 背景：浅灰色弧线
  - 进度弧：彩色（根据分数变化）
  - 直径：200px
  
#### 中心文本
- **分数**: 87（超大号数字）
- **等级**: "健康"（彩色文字）

#### 颜色编码
- 80-100 分：绿色
- 60-80 分：黄色
- 40-60 分：橙色
- 0-40 分：红色

#### 交互
- 悬停显示评分维度详情（工具提示）

### 4. 知识增长趋势图（右侧 2/3 宽度）

#### 图表控制区
- **时间切换按钮**: [7 天] [30 天] [90 天]
- **活动状态**: 当前选中按钮高亮（蓝色）

#### 图表区域
- **类型**: SVG 面积折线图
- **曲线**: 平滑过渡
- **填充**: 渐变蓝色（上深下浅）
- **数据点**: 圆形标记
- **坐标轴**: 
  - X 轴：日期标签
  - Y 轴：数量刻度

#### 交互
- 鼠标悬停显示具体数值
- 点击数据点高亮显示

### 5. 领域覆盖热力图（右下）

#### 列表项结构
```
[领域名称] [进度条        ] [百分比]
   AI       ████████░░      85%
 数据库     ██████░░░░      75%
 网络       ██░░░░░░░░      25%
```

#### 进度条颜色
- >80%: 深绿色
- 60-80%: 浅绿色
- 40-60%: 黄色
- <40%: 橙色

#### 交互
- 悬停显示精确百分比和知识数量
- 点击展开子领域（待实现）

### 6. 知识质量雷达图（左下）

#### 五维度
1. **新鲜度** - 知识更新及时性
2. **覆盖度** - 领域覆盖全面性
3. **连通性** - 知识关联紧密度
4. **准确性** - 内容事实正确性
5. **多样性** - 知识类型丰富度

#### 图表结构
- **背景网格**: 5 层同心多边形
- **轴线**: 从中心辐射的 5 条线
- **数据多边形**: 蓝色半透明填充
- **标签**: 维度名称标注在顶点外

#### 评分
- 5 星制或百分制
- 示例：新鲜度 4.5/5.0

### 7. 更新时间线（底部全宽）

#### 事件项结构
```
● 时间戳 [类型图标] 事件标题
  事件描述
  [统计徽章]
```

#### 时间点样式
- **实时更新**: 蓝色圆点 🔵
- **定时任务**: 绿色圆点 🟢
- **事件触发**: 橙色圆点 🟠

#### 连接线
- 垂直灰色细线贯穿所有事件
- 每个事件左侧对齐时间线

#### 统计徽章
- **新增**: 绿色 `+23`
- **修改**: 黄色 `~56`
- **删除**: 红色 `-12`

#### 交互
- 点击"查看更多"加载更多历史
- 点击事件展开详细变更列表

## 响应式适配

### 桌面端 (>1200px)
- 完整网格布局
- 所有组件并排显示

### 平板端 (768px-1200px)
- 健康仪表盘和增长图全宽显示
- 热力图和雷达图并排
- 时间线保持全宽

### 移动端 (<768px)
- 单列堆叠布局
- 每个组件占一行
- 垂直滚动查看
